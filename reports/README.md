# Reports Directory

This directory contains audit reports generated by AuditMySit.

**Note:** This directory is excluded from Git (.gitignore)

## Contents

Reports are generated in various formats:
- JSON (*.json) - Machine-readable reports
- HTML (*.html) - Interactive dashboard reports
- Markdown (*.md) - Human-readable summaries

## Usage

```bash
# Generate reports
auditmysit https://example.com -f json -o reports/example.json
auditmysit https://example.com -f html -o reports/example.html
```

## Cleanup

```bash
# Remove all reports
rm -rf reports/*

# Remove specific report
rm reports/example.json
```
